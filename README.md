# Little Lemon Restaurant

A modern, accessible restaurant website for Little Lemon, a family-owned Mediterranean restaurant in Chicago. The site allows customers to easily browse the menu, view weekly specials, read testimonials, and make online table reservations.

## Features

- **Online Reservations**: Book a table with our easy-to-use reservation form
- **Weekly Specials**: Browse this week's featured Mediterranean dishes
- **Customer Testimonials**: See what our guests are saying
- **Responsive Design**: Optimized for both desktop and mobile devices
- **Fully Accessible**: Built with WCAG standards and semantic HTML for screen readers

## Tech Stack

- React + Vite
- Material UI (MUI)
- React Router
- Yup validation
- Vitest for testing

## Getting Started

```bash
npm install
npm start
```

Visit `http://localhost:5173` to view the website.
